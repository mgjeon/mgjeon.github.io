Field line of a vector field

A field line of a vector field is a curve where the vector field’s direction is the same as the curve’s tangent at every point.

For a vector field \mathbf{F}(\mathbf{r}) defined at every point \mathbf{r}, the equation of a field line can be expressed as

\frac{\rm{d}\mathbf{r}}{|\rm{d}\mathbf{r}|} = \frac{\mathbf{F}(\mathbf{r})}{|\mathbf{F}(\mathbf{r})|}

For simplicity, we often use \rm{d}s = |\rm{d}\mathbf{r}| and F = |\mathbf{F}(\mathbf{r})|.

\frac{\rm{d}\mathbf{r}}{\rm{d}s} = \frac{\mathbf{F}}{F}

2D vector field

The 2D vector field in Cartesian coordinates (x, y) is

\mathbf{F}(x, y) = F_x(x, y)\mathbf{\hat{x}} + F_y(x, y)\mathbf{\hat{y}}

and the displacement vector \rm{d}\mathbf{r} is

\rm{d}\mathbf{r} = \rm{d}x \mathbf{\hat{x}} + \rm{d}y \mathbf{\hat{y}}

Then the field line equations for each component read

\begin{cases} \displaystyle \frac{\rm{d}x}{\rm{d}s} & = \displaystyle \frac{F_x}{F} \\ \\ \displaystyle \frac{\rm{d}y}{\rm{d}s} & = \displaystyle \frac{F_y}{F} \end{cases}

In the following examples, the vector field is given by

\mathbf{F}(x, y) = x\mathbf{\hat{x}} + (x+y)\mathbf{\hat{y}}

from scipy.integrate import solve_ivp

def fx(x, y):
    return x 

def fy(x, y):
    return x + y

def dSdt(t, S):
    x, y = S
    Fx = fx(x, y)
    Fy = fy(x, y)
    F = np.sqrt(Fx**2 + Fy**2)
    dx_dt = Fx/F 
    dy_dt = Fy/F 
    return [dx_dt, dy_dt]

def boundary(t, S):
    if np.abs(S[0]) > 12 or np.abs(S[1]) > 12:
        return 0
    else:
        return -1

boundary.terminal = True

plt.figure(figsize=(10, 10))
plt.gca().set_aspect('equal')
plt.axvline(0, color='k')
plt.axhline(0, color='k')
plt.xlim(-12, 12)
plt.ylim(-12, 12)

t_span = [0, 1000]
t_eval = np.linspace(t_span[0], t_span[1], 10000)
for S0 in r:
    sol = solve_ivp(dSdt, t_span, S0, t_eval=t_eval, events=boundary)
    xs, ys = sol.y
    plt.plot(xs, ys, color='blue')

plt.scatter(r[:, 0], r[:, 1])

PyVista (+ K3D-jupyter)

import pyvista as pv 
pv.set_jupyter_backend('static')

def create_coordinates(bounds):
    xbounds = (bounds[0], bounds[1])
    ybounds = (bounds[2], bounds[3])
    zbounds = (bounds[4], bounds[5])
    meshgrid = np.mgrid[xbounds[0]:xbounds[1]+1, ybounds[0]:ybounds[1]+1, zbounds[0]:zbounds[1]+1]
    return np.stack(meshgrid, axis=-1).astype(np.float32)

def create_mesh(Fx, Fy, Fz):
    Nx, Ny, Nz = Fx.shape
    co_bounds = (0, Nx-1, 0, Ny-1, 0, Nz-1)
    co_coords = create_coordinates(co_bounds).reshape(-1, 3)
    co_coord = co_coords.reshape(Nx, Ny, Nz, 3)
    x = co_coord[..., 0]
    y = co_coord[..., 1]
    z = co_coord[..., 2]
    mesh = pv.StructuredGrid(x, y, z)
    vectors = np.stack([Fx, Fy, Fz], axis=-1).transpose(2, 1, 0, 3).reshape(-1, 3)
    mesh['vector'] = vectors
    mesh.active_vectors_name = 'vector'
    magnitude = np.linalg.norm(vectors, axis=-1)
    mesh['magnitude'] = magnitude
    mesh.active_scalars_name = 'magnitude'
    return mesh
